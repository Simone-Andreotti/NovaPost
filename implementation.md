2. Tech Stack & Dependencies
The requirements.txt should strictly include:

openai: To interact with the Nvidia NIM API (e.g., meta/llama-3.1-405b-instruct or mistralai/mistral-large) using their OpenAI-compatible endpoint.

python-dotenv: To load the .env file securely.

edge-tts: For local, free text-to-speech generation.

faster-whisper: For fast, local subtitle generation.

ffmpeg-python: A Python wrapper for FFmpeg.

Pillow (PIL): For image manipulation, applying the dark space overlays, and drawing the cyan typography.

pypdf: For merging multiple processed images into a single LinkedIn PDF carousel.

Step 1: Asset Discovery (def find_assets())
Scan the assets/ folder.

Locate exactly one .mp4 file. If none or multiple are found, raise an error.

Locate all .png and .jpg files. Sort them alphabetically to maintain order. Count the total number of images found.

Read the contents of prompt.md.

Step 2: AI Content Generation (def generate_copy(prompt_text, image_count))
Initialize the openai client with the NVIDIA_NIM_API_KEY and Nvidia's base URL (https://integrate.api.nvidia.com/v1).

Create a system prompt instructing the AI to act as a creative front-end developer specializing in custom 3D animations, semantic HTML, and web performance.

Pass the prompt_text and explicitly state the image_count.

Force the LLM to return a strict JSON object with the following schema:

x_post: (string) A short, engaging post for X under 280 characters, including tags like #Threejs, #WebGL.

tiktok_script: (string) A spoken script designed to be engaging and under 30 seconds.

linkedin_slides: (array of strings) An array where the length exactly matches the image_count. Each string should be 1-2 short sentences summarizing a key technical or performance point.

Step 3: X Post Generation
Extract x_post from the JSON.

Save the string to output/x_post.txt.

Step 4: TikTok Video Generation (def build_tiktok_video(video_path, script_text))
TTS: Use edge-tts to read the script_text and save it to output/voiceover.mp3.

Transcription: Load the faster-whisper base model. Transcribe voiceover.mp3 and save the output as output/subtitles.srt.

Video Processing (FFmpeg): Use ffmpeg-python to execute the following pipeline using the local bin/ffmpeg.exe:

Take the input video.mp4.

Crop the video to a 9:16 vertical ratio (centered on the 3D element).

Loop or trim the video to match the exact duration of voiceover.mp3.

Replace the original audio with voiceover.mp3.

Burn the subtitles.srt into the center of the video using a clean, bold font.

Export as output/tiktok_final.mp4.

Step 5: LinkedIn Carousel Generation (def build_linkedin_carousel(image_paths, slide_texts))
Image Processing: Loop through each image path and its corresponding text from the linkedin_slides array.

For each image using Pillow:

Apply a semi-transparent dark overlay (e.g., black with 50% opacity) to mimic the dark space theme of the portfolio.

Dynamically wrap the text to fit the image width.

Draw the wrapped text in the center of the image. Visual constraint: Use a modern, technical sans-serif font colored in the specific cyan hex code from the portfolio's "SIMONE ANDREOTTI" branding.

Save the modified image temporarily in output/.

PDF Generation: Use pypdf.PdfWriter to merge all the newly processed images in sequential order into a single document.

Save the final document as output/linkedin_carousel.pdf. Ensure images are compressed to keep the final PDF under 25MB.

Clean up the temporary modified images.

4. Main Execution Block
Ensure the script runs automatically when executed, running through Steps 1-5, printing success messages to the console upon completing each phase.
